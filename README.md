## Description

Install UMD4/UMD5 repositories.

## Setup

This module is available on puppet forge:

```
puppet module install cnafsd-umd
```

## Usage

Use this module as follow:

```
include umd::umd5
```

## Limitations

It works only on RedHat CentOS 7 or Alma Linux 9 distributions.
